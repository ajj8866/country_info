Neighbours Yields neighbours of countries user enters
User input:
Country Name
Weather Show list of weather stations and their weather observations within coordinates input. List of user inputs
  1. North: Minumum value of 0 and maximum value of 90
  2. South: Minimum value of -90 and maximum value of 0
  3. East: Minimum value of -180 and maximum value of 180
  4. West: Minimum value of -180 and maximum value of 180
  5. Max Rows
Timezone Shows timezone ID, GMT offset, sunrise and sunset for user input longitude and lattitude
  1. Longitude: Between -90 and 90 (positive values for northern hemisphere and negative values for southern hemisphere)
  2. Lattitude: Between -180 and 180 with 0 being the prime meridian (Greenwich, England), positive values being east of prime meridian and negative values west of prime meridian
